# ReadMe Group Assignment

### Focus for this project was soley on agile methods and working together as a team with different dev branches and feature branches

The idea was to create a web page/register hybrid UI with a cloud db for a hypothetical customer in the form of a retro vhs rental store.
<br> My main focus was on being the Scrum Master while also contributing with some code and reviewing pull requests.
<br> We worked with sprints and a backlog etc on Jira to help add structure and the project overview.

The actual project was never fully finished before we had to move on to the next school course, as seen by the fact that there's a waiting PRs. 
Finishing was actually not a requirement for the course, learning to work together as a agile team was, but ofcourse I would have wanted to finish it all.
